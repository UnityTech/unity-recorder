    Class RecorderBindings

    Helper component mainly used to save Recorder's GameObject references. Some recorders (like the Animation Recorder) requires a GameObject reference from a Scene to record from. In this cases, this component is automatically added to the Scene and binds the selected GameObject to the recorder settings.
